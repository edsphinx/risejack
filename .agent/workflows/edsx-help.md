# /edsx help

## Available Commands

| Command        | Description                                 |
| -------------- | ------------------------------------------- |
| `/edsx load`   | Load project context (run at session start) |
| `/edsx save`   | Save session summary to chat history        |
| `/edsx status` | Check MCP server status                     |
| `/edsx help`   | Show this help                              |

## Quick Start

1. Start session: `/edsx load`
2. Do your work
3. End session: `/edsx save`

## MCP Server

- Location: `~/.mcp/servers/risecasino/`
- Port: 3100
- Auto-starts on login via systemd

## Context Hub

- Location: `~/Blockchain/RiseJack (Mobile)/ai-context-hub/`
- Contains: preferences, project contexts, chat history
