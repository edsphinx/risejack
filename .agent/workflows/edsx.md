# /edsx - Command Hub

## Available Subcommands

Type the full command:

| Command        | Description                       |
| -------------- | --------------------------------- |
| `/edsx-load`   | Load project context (run first!) |
| `/edsx-save`   | Save session to history           |
| `/edsx-status` | Check MCP server status           |
| `/edsx-help`   | Show all commands                 |

## Quick Reference

```
Session start:  /edsx-load
Session end:    /edsx-save
```
